## 접근
아나그램은 문자열 내 문자로 문자열을 재구성하므로
문자열 내 문자의 종류를 구해 일치하는지를 반환하면 정답일 것이다.

### 제출코드
```swift
class Solution {
    func isAnagram(_ s: String, _ t: String) -> Bool {
        return Set(s) == Set(t)
    }
}
```
### 제출 결과
#Input
```
s =
"aa"
t =
"a"
```
#Output
true
#Expected
false

### 분석
아나그램은 문자의 종류 뿐만 아니라 각 문자의 수 또한 같아야 함
